Panduan Pasang Aplikasi di HP

🔗 hadiran-rt.vercel.app
🤖 HP Android Chrome
1Buka link di atas dengan Chrome
2Muncul tombol Pasang → ketuk
3Kalau tidak muncul: menu ⋮ → Instal aplikasi
4Ketuk Instal ✅ ikon muncul di layar
🍎 iPhone / iPad Safari
1Buka link di atas dengan Safari
2Ketuk ikon Bagikan ⬆️ (kotak panah ke atas)
3Pilih Tambahkan ke Layar Utama
4Ketuk Tambah ✅ ikon muncul di layar
⚠️ Jangan buka dari dalam WhatsApp. Ketuk titik tiga ⋮ di pojok kanan atas → Buka di browser dulu, baru ikuti langkah di atas.
